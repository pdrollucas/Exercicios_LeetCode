# Exercicios_LeetCode

**Disciplina:** Algoritmos avançados

**Alunos:** Eduardo Correa, Eduardo Fritz e Pedro Lucas Luckow

**Explicação do repositório:** Há três pastas: Array exercises, Linked Lists exercises e Stack exercises. Em cada uma dessas pastas há a divisão de dificuldades: easy ones e medium ones. Em cada um dos arquivos há o seu respectivo link do exercício. Foi utilizada unicamente a plataforma LeetCode. Foram utilizadas as linguagens JavaScript, TypeScript e Python, escolhas pessoais de cada aluno.
